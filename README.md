Minha experiência criando este projeto foi muito produtiva e focada.
Embora eu já tivesse a base pronta, meu trabalho principal foi seguir um padrão consistente e organizar o projeto
de forma clara e eficiente.
Durante o processo, organizei as DTOs para estruturar e transferir os dados de forma segura e consistente entre as camadas, 
as interfaces funcionaram como contratos claros entre services e repositórios, 
enquanto os services centralizaram a lógica de negócio e os validadores garantiram que os dados de entrada fossem corretos antes de qualquer processamento.
Também dediquei atenção à infraestrutura do projeto, mantendo pastas e módulos organizados, o que tornou o código mais claro e fácil de manter. Essa experiência reforçou minha disciplina, 
atenção aos detalhes e compreensão de boas práticas de arquitetura de software.
